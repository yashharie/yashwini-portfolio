## 🛠️ Installation

Clone the repository:

```bash
git clone https://github.com/yashharie/yashwini-portfolio-.git
```

Navigate to the project:

```bash
cd yashwini-portfolio-
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
